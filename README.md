##GraphQL Client

### Steps to Run
- > Checkout to corresponding [server](https://github.com/manojd929/graphql-ecommerce-server) folder
- Start the server:
    > npm install
    > npm start 

- Add client_id in [index.html](/public/index.html)
- Start the client
    > npm install
    > npm start
